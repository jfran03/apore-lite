# Venn Diagrams

## Definition
A Venn diagram pictures sets represented as circles. The way the circles overlap (or don't) shows the relationships between the sets.
> Source: Venn Diagrams.html

## Key Concepts

### Intersection
The intersection of two sets is the set of elements they both have in common.
- **Notation:** A ∩ B
- **Definition:** A ∩ B = { x | x ∈ A and x ∈ B }
- **On a Venn diagram:** the overlapping region of the two circles.
  > Source: Venn Diagrams.html

**Example:** A = {1,2,3}, B = {3,4,5} → A ∩ B = {3}
> Source: Venn Diagrams.html

### Union
The union of two sets is the set of elements that are in either set (or both).
- **Notation:** A ∪ B
- **Definition:** A ∪ B = { x | x ∈ A or x ∈ B }
- **On a Venn diagram:** the entire area covered by both circles combined.
  > Source: Venn Diagrams.html

**Example:** A = {1,2,3}, B = {3,4,5} → A ∪ B = {1,2,3,4,5}
> Source: Venn Diagrams.html

### Disjoint Sets (on Venn Diagrams)
Two sets are disjoint if they share no elements (A ∩ B = ∅). On a Venn diagram, the circles don't touch.
> Source: Venn Diagrams.html

**Example:** A = {1,2,3}, B = {4,5,6} → disjoint
> Source: Venn Diagrams.html

## Rules for Reading Venn Diagrams
- If two circles don't overlap, the sets are disjoint.
- The overlapping part represents the intersection.
- Shading both circles together represents the union.
  > Source: Venn Diagrams.html

## Historical Note
Venn diagrams are named after English logician John Venn (1834–1923), who introduced them in 1880 to visually represent logical relations between sets.
> Source: Venn Diagrams.html

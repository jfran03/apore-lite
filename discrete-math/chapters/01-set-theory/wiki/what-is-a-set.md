# What is a Set?

## Definition
A set is an unordered collection of distinct objects, treated as a single entity. The objects inside the set are called its elements or members.
> Source: What is a Set.html

## Key Concepts
- **Unordered:** The order of elements doesn't matter. {2,4,6,8} is the same set as {8,6,4,2}. This is different from a list or an array in programming, where order does matter.
  > Source: What is a Set.html
- **Collection:** A set groups items together into a single whole; you reason about the group as one entity.
  > Source: What is a Set.html
- **Distinct:** Each element in a set must be unique. Duplicates don't count — {red, blue, blue, yellow} is really just {red, blue, yellow}.
  > Source: What is a Set.html
- **Elements:** The things inside a set can be numbers, names, processes, IP addresses, or even other sets. Each element must be well-defined so you can clearly decide whether it belongs in the set.
  > Source: What is a Set.html
- **Notation:** A set is usually represented with a capital letter (like A, B or C) and its elements appear inside curly braces { }.
  > Source: What is a Set.html

## Examples
- A = {Alex, Tippy, Shells, Shadow} (a set of names)
- B = {red, blue, yellow} (a set of primary colours)
- C = {{a, b}, {a, c}, {b, c}} (a set made up of smaller sets)
  > Source: What is a Set.html

## Valid vs Invalid Sets
A set must be well-defined — you must be able to clearly decide whether something belongs in it.
- ✅ The group of all students enrolled in MATH 3000 — well-defined, this is a set.
- ❌ The set of "beautiful colours" — not a set because "beautiful" is subjective.
- ✅ The numbers {2, 4, 6, 8} — well-defined, this is a set.
  > Source: What is a Set.html

## Common Number Sets
| Symbol | Name | Members |
|--------|------|---------|
| ℕ | Natural numbers | {0, 1, 2, 3, …} |
| ℤ | Integers | {…, −2, −1, 0, 1, 2, …} |
| ℚ | Rational numbers | fractions like ½, ¾ |
| ℝ | Real numbers | all decimals and fractions |
| ℂ | Complex numbers | numbers with i |

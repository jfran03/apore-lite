# Ways to Describe Sets

## Definition
Two main methods are used to describe sets: the Roster Method and Set-Builder Notation.
> Source: Ways to Describe Sets.html

## Key Concepts

### Roster Method
The roster method lists every element of the set, separated by commas, inside curly braces { }.
- Use roster when the set is small and easily listed.
  > Source: Ways to Describe Sets.html

### Set-Builder Notation
Set-builder notation describes a set by stating the property or rule its elements must satisfy. General form:

{ x | condition on x }

- Use set-builder when the set is large or defined by a clear rule.
  > Source: Ways to Describe Sets.html

Neither method is better than the other — each is useful in different contexts.
> Source: Ways to Describe Sets.html

## Examples

### Roster Method
- A = {2, 4, 6, 8}
- B = {red, blue, yellow}
  > Source: Ways to Describe Sets.html

### Set-Builder Notation
- { x ∈ ℕ | x < 5 } → "the set of natural numbers less than 5" = {0, 1, 2, 3, 4}
- { x ∈ ℤ | x is even } → "the set of even integers"
  > Source: Ways to Describe Sets.html
